评估方法说明

DOM 结构对比

逐层对比原始页面与复刻页面的 DOM 树结构，分析标签层级、节点数量、属性差异，计算结构相似度得分。支持忽略白空间节点和注释节点的智能匹配。

CSS 属性对比

提取计算后的 CSS 样式，逐属性对比颜色值、间距、字号、圆角等视觉属性，以加权欧氏距离衡量差异程度，输出属性级匹配率。

像素级视觉对比

对原始与复刻页面进行同分辨率截图，通过 SSIM 算法进行像素级相似度分析，识别视觉偏移区域并生成差异热力图。

功能测试用例覆盖率

基于预定义交互测试用例，自动化验证复刻页面的功能行为是否与原始页面一致，包括表单提交、按钮响应、页面跳转等核心流程。
