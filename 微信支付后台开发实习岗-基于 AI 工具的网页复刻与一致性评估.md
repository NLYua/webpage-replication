# 项目考察：基于 AI 工具的网页复刻与一致性评估

## 1. 问题描述

请使用 AI 工具（如 CodeBuddy、Cursor、Claude Code、Deepseek/OpenAI/Claude API 等），完成以下两部分工作：

### 1.1 网页复刻

给定一个目标网址，并指定需要复刻的功能范围，使用 AI 工具生成可运行的复刻页面。

**示例场景：**

- **场景一：百度首页**
  - 网址：`https://www.baidu.com`
  - 复刻范围：搜索输入框、搜索按钮点击交互、搜索结果文案展示、结果列表翻页功能。

- **场景二：微信支付商户登录页**
  - 网址：`https://pay.weixin.qq.com/index.php/core/home/login`
  - 复刻范围：仅复刻用户名、密码、验证码输入区域，以及登录按钮的点击交互。

**要求：**

- 复刻产物需可在本地或线上独立运行，并能体现指定的核心功能与交互逻辑。
- 复刻产物要有完整的可长期维护的源码
- 对同类型的网站复刻效果稳定，在另一个新的页面上除了给出初始需求外，无需额外干预AI。

### 1.2 一致性评估

对复刻结果与原始页面进行对比分析，输出一份评估报告，覆盖以下维度：

- 功能一致性：指定功能是否完整实现且行为正确。
- 交互一致性：用户操作流程、反馈、状态变化是否与原页面一致。
- 视觉一致性：布局、配色、字体、间距、组件样式等还原度。

**要求：**

- 要有明确的可量化、可置信的度量指标
- 复刻后，自动评估，无需额外人工驱动
- **加分项**：更丰富的度量指标证明复刻质量
