# AI Prompt 记录

## 复刻页面 Prompt
- 场景：百度首页复刻
- 要求：复刻搜索输入框、搜索按钮、搜索结果展示、结果分页交互，保留首页顶部导航和热搜列表样式。

- 场景：微信支付商户登录页复刻
- 要求：复刻用户名、密码、验证码输入区、登录按钮、密码显隐、验证码刷新、登录交互和提示。

- 场景：知乎登录页复刻
- 要求：复刻手机号登录/密码登录切换、验证码发送倒计时、密码显隐、第三方登录入口、品牌展示区和页面布局。

## 一致性评估 Prompt
- 目标：对比原始页面和复刻页面，输出视觉、功能、交互三个维度的量化评分。
- 输出格式：严格返回 JSON，包括 `visual`、`functional`、`interaction`、`overall`、`summary`。
- 维度细分：DOM结构一致性、CSS属性匹配、布局相似度、色彩还原度、功能覆盖率、功能正确率、数据完整性、交互还原度、反馈一致性、状态管理。
