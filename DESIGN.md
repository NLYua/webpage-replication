# 文档风

## 类型
开发者文档、API 平台、开源工具、CLI 工具、技术产品说明。

## 设计关键词
深色、代码、网格、快速检索、开发者友好、低摩擦。

## 色彩
- 背景：#08111F
- 面板：#0E1A2D
- 代码绿：#62F6A8
- 链接蓝：#7AB7FF
- 文本：#E6EDF7
- 次文本：#8EA0B8
- 边框：#1D2B44

## 字体
正文使用 Inter，代码使用 JetBrains Mono、SFMono 或 Consolas。

## 布局
左侧文档导航，右侧内容区域。首页突出快速开始和代码示例。

## 组件
- 左侧目录
- 代码块
- API 方法标签
- 快速开始卡
- 版本提示

## 动效
动效极少，重点保证阅读和复制效率。

## 禁忌
不要用花哨营销插图，不要降低代码可读性。
