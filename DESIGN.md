# 浅色文档风

## 类型
开发者文档、API 平台、开源工具、CLI 工具、技术产品说明。

## 设计关键词
浅色、清晰、数据驱动、开发者友好、低摩擦。

## 色彩
- 背景：#FFFFFF
- 面板：#FFFFFF
- 主色绿：#16A34A
- 链接蓝：#3B82F6
- 文本：#1E293B
- 次文本：#64748B
- 边框：#E2E8F0

## 字体
正文使用 Inter，代码使用 JetBrains Mono、SFMono 或 Consolas。

## 布局
左侧文档导航，右侧内容区域。首页突出快速开始和代码示例。

## 组件
- 左侧目录
- 代码块
- API 方法标签
- 快速开始卡
- 版本提示

## 动效
动效极少，重点保证阅读和复制效率。

## 禁忌
不要用花哨营销插图，不要降低代码可读性。
